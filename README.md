# Blog

Personal blog

## Installation

```sh
$ brew install automake
```

```sh
$ npm config set registry https://registry.npm.taobao.org
```

```sh
$ npm install -g hexo-cli
```

```sh
$ git pull <this repo>
```

```sh
$ cd <this repo>
$ rm -rf node_module # 如果需要的话
$ npm install
```

## Develop

```sh
$ make run
```
