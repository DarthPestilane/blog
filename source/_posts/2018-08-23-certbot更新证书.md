一键更新证书的脚步，适用于docker容器中的nginx

```sh
#!/bin/sh

# usage:
# ./cert.sh <certonly|renew>

docker-compose stop nginx

docker run -it --rm --name certbot \
            -v "/etc/letsencrypt:/etc/letsencrypt" \
            -v "/var/lib/letsencrypt:/var/lib/letsencrypt" \
            -p 80:80 \
            -p 443:443 \
            certbot/certbot "$@"

docker-compose up -d
```
